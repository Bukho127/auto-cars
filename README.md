# Auto Cars (React + Vite)

A responsive car showroom UI app built with React, Vite, Redux Toolkit and React Router. Optimized for a desktop + mobile demo for a university assignment.

---


## 🛠 Installation

1. `npm install`
2. `npm run dev`
3. Open `http://localhost:5173`

---

## 📦 NPM scripts

- `npm run dev` - Run development server
- `npm run build` - Build production bundle
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint checks (if configured)

---

## 🧩 Tech stack

- React 18
- Vite
- Redux Toolkit (store slice)
- React Router v6
- ESLint
- CSS (global + component styling)

---

## 🎯 User workflow

1. Open homepage
2. Browse products on `ProductPage`
3. Click product card to open `ProductDetailsPage`
4. Add item to cart
5. Manage cart on `CartPage` and checkout flow (UI-only)
